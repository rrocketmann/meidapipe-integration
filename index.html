About Hanvas

Hanvas is a real-time browser drawing app powered by MediaPipe hand landmark tracking. The webcam stream is analyzed frame-by-frame, and a 21-point hand skeleton is estimated in normalized coordinates, then mapped to the drawing canvas.

Drawing behavior is gesture-gated: closed or partially closed hands create paint points, and nearby sampled points are interpolated to reduce visible gaps and produce smoother strokes. Paint color is selected in the toolbar and stored per-point so mixed-color drawings keep their original segments.

Libraries used: MediaPipe Tasks Vision (`HandLandmarker`, `FilesetResolver`) for detection, plus MediaPipe drawing utilities for landmark rendering overlays.

Export captures the active drawing canvas as PNG. Project source is MIT licensed; bundled MediaPipe components are provided under Apache License 2.0.
